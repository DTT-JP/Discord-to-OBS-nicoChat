# Discord-to-OBS-nicoChat

Discord のメッセージを OBS に送信する Discord Bot です。

## 動作環境

- Node.js 22.0.0 以上
- OBS ブラウザソースに表示される URL の有効期限は 60秒以内
- メッセージのコメントは文字数が多いほど速く流れます

## コマンド一覧

### `/start #チャンネル`

OBS オーバーレイのセッションを開始します。

DM に URL を送信してください。

### `/auth [上記の6桁コード]`

OBS ブラウザに表示されたコードを入力して認証を完了します。

### `/status`

CPU 使用率とアクティブセッション数を表示します。

## セッションの終了

セッションの終了は Discord Bot が自動的に行われます。

セッションの終了には、メッセージのコメントが残ります。
