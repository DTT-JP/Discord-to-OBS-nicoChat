# メッセージ表示・メタデータ・セッション

オーバーレイ表示ルールの利用者向けガイドです。  
最終仕様は `discord/parser.js` と `public/script.js` を正とします。

## メタデータ書式

メッセージ中の `?` から `?` までを属性として解釈します。  
属性は本文の先頭・末尾・途中のどこに書いても有効です。

```text
?属性1 属性2? 本文
本文 ?属性1 属性2?
本文 ?属性? 続き
```

- 属性は半角スペース区切りで複数指定できます
- 属性の順序は自由です

### 色

| 値 | 色 |
|---|---|
| `white` | 白 (`#FFFFFF`) |
| `red` | 赤 (`#FF4040`) |
| `pink` | ピンク (`#FF80C0`) |
| `yellow` | 黄 (`#FFE133`) |
| `orange` | オレンジ (`#FF9933`) |
| `green` | 緑 (`#33DD44`) |
| `cyan` | シアン (`#33DDFF`) |
| `blue` | 青 (`#4488FF`) |
| `purple` | 紫 (`#AA44FF`) |
| `black` | 黒 (`#111111`) |
| `#RRGGBB` | 任意色（例: `#ff8800`） |

### サイズ

| 値 | サイズ |
|---|---|
| `big` | 12vh |
| `medium` | 6vh（既定） |
| `small` | 3vh |

Discord の見出し書式もサイズ指定として反映されます。

| Discord書式 | 反映サイズ |
|---|---|
| `# テキスト` | `big` |
| `## テキスト` / `### テキスト` | `medium` |
| `-# テキスト` | `small` |

メタデータ指定がある場合は、そちらが優先されます。

### 位置

| 値 | 効果 |
|---|---|
| `ue` | 画面上部に固定表示 |
| `shita` | 画面下部に固定表示 |

指定なしの場合は横スクロール表示になります。  
固定表示コメントは個別タイマーで消去され、他行に影響しません。

### 記述例

```text
?red ue? お知らせ
?blue big? **重要** テキスト
テキスト ?shita green?
?#ff8800 big ue? カスタムカラー
```

## テキスト装飾

Discord の装飾記法がそのまま反映されます。

| 書式 | 効果 |
|---|---|
| `**テキスト**` | 太字 |
| `*テキスト*` | 斜体 |
| `__テキスト__` | 下線 |
| `~~テキスト~~` | 取り消し線 |

複数組み合わせも可能です（例: `**__太字下線__**`）。  
改行（Shift+Enter）や AA 表示にも対応します。

## 絵文字・スタンプ

- カスタム絵文字（`<:name:id>`）は OBS で画像表示されます
- Unicode 絵文字（😊 など）は文字として表示されます
- 絵文字サイズはテキストサイズに連動します

## セッション関連

### 同時表示上限（開始時）

```text
/start channel:#チャンネル名 limit:100
```

`limit` 未指定時は `.env` の `MAX_COMMENTS` が適用されます（範囲: 1〜99999）。

### 配信中の設定変更

```text
/session limit:50
/session secret:deny
```

`/session` はグローバルBL対象外ユーザーが実行でき、対象は自分が作成したセッションのみです。

### 認証コマンド

`/auth` は DM ではなく、`/start` と同じサーバーのチャンネルで実行してください。
